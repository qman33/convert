# MP3 to OGG

Convert a single `.mp3` file to `.ogg` in the same folder.

## Setup

```cmd
pip install -r requirements.txt
```

Requires Python on your PATH.

## Usage

```cmd
convert-mp3-to-ogg C:\path\to\file.mp3
```

Run from this folder, or pass the full path to `convert-mp3-to-ogg.cmd`.

Output: `file.ogg` next to `file.mp3`.
